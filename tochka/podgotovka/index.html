Подготовка к воркшопу

Соберём рабочее место
за 15 минут

На воркшопе мы будем собирать ваше предложение с помощью ИИ прямо в редакторе. Чтобы не терять время на эфире — поставьте два бесплатных инструмента заранее по шагам ниже.

🧩 VS Code ✱ Claude Code ⏱️ ~15 минут 💻 Windows и macOS

Зачем это нужно

Коротко, чтобы было спокойно ставить.

VS Code — бесплатный редактор от Microsoft. Мы используем его не как «программу для программистов», а как удобное окно, где ИИ-помощник видит ваши файлы и складывает в них готовые материалы.

Claude Code — ИИ-помощник от Anthropic, который встраивается внутрь VS Code. Это он на воркшопе будет задавать вопросы про ваш бизнес и собирать предложение. Терминал и навыки программирования не нужны — только переписка в чате.

Что понадобится заранее

  • 🔑
    Подписка Claude Pro или Max. Без неё в Claude Code не войти. Оформляется на claude.ai, оплата картой. Сделайте это до воркшопа.
  • 💻
    Компьютер (Windows или Mac). Со смартфона/планшета не получится — нужен «настольный» компьютер или ноутбук.
  • ⏱️
    15 свободных минут и интернет. Лучше не за 5 минут до эфира — поставьте спокойно заранее.
Важно про доступ. Сервисы Anthropic в России работают нестабильно — возможно, понадобится стабильное соединение, чтобы оформить подписку и войти. Если на шаге входа возникнут сложности — напишите нам, поможем заранее, а не на эфире.

Шаг 1 · Установить VS Code

Выберите свою систему:

1

Скачать установщик

  1. Откройте сайт code.visualstudio.com.
  2. Нажмите синюю кнопку «Download for Windows» — скачается файл вида VSCodeUserSetup-….exe.
2

Запустить и установить

  1. Откройте скачанный файл, примите соглашение, нажимайте «Далее».
  2. На шаге с галочками оставьте включённой «Add to PATH» и можно отметить «Открывать с помощью Code» — затем «Установить».
  3. В конце нажмите «Завершить» — VS Code откроется сам.
Готово, когда: открылось тёмное окно VS Code с приветствием.
1

Скачать приложение

  1. Откройте сайт code.visualstudio.com.
  2. Нажмите кнопку «Download for macOS» — скачается архив.
2

Перенести в «Программы»

  1. Откройте скачанный файл и перетащите «Visual Studio Code» в папку «Программы» (Applications).
  2. Запустите VS Code из «Программ». Если спросит «открыть приложение из интернета» — подтвердите.
Готово, когда: открылось тёмное окно VS Code с приветствием.

Шаг 2 · Установить помощника Claude Code

1

Открыть «Расширения»

Слева в VS Code — вертикальный ряд иконок. Нажмите «Extensions» (квадратик из кубиков) или сочетание клавиш:

Windows: Ctrl + Shift + XMac: Cmd + Shift + X

2

Найти и установить

  1. В строке поиска наберите Claude Code.
  2. Выберите расширение Claude Code от Anthropic (с галочкой-значком издателя) → Install.
  3. Если попросит — перезапустите редактор (CtrlCmd + Shift + P → наберите Reload Window → Enter).
Альтернатива: откройте в браузере ссылку vscode:extension/anthropic.claude-code — она сама откроет VS Code на странице расширения.
Готово, когда: у «Claude Code» в расширениях появилась кнопка «Uninstall» — значит установлено.

Шаг 3 · Войти по подписке

1

Открыть панель Claude Code

Нажмите значок Claude Code (✱) — он появляется в боковой панели слева или в строке состояния внизу окна.

2

Sign in

  1. Нажмите Sign in — откроется браузер.
  2. Войдите в свой аккаунт Claude (тот, где оформлена Pro или Max) и подтвердите вход.
  3. Вернитесь в VS Code — повторно входить больше не нужно.
Готово, когда: в панели Claude Code появилось поле для ввода — чат готов принять сообщение.

Шаг 4 · Проверить, что всё работает

В поле Claude Code напишите любое сообщение, например:

Привет! Ты готов помочь мне на воркшопе?

Если пришёл осмысленный ответ — вы готовы к воркшопу 🎉. Ничего больше настраивать заранее не нужно.

Совет по качеству (по желанию). В меню режимов панели (вызывается Shift + Tab) можно выбрать модель Opus и уровень усилий High — ответы будут глубже. Это не обязательно для старта.

Чтобы ИИ собрал сильное предложение — ему нужен контекст про ваш бизнес. Соберите заранее:

🏢

Чем занимаетесь, какой продукт/услуга, для кого, сколько лет на рынке.

🎯

Не нужно ничего красиво оформлять — достаточно заметок в свободной форме. Если есть сайт или прайс — тоже пригодятся, можно дать на них ссылку прямо в чате.

Частые вопросы

Нужно ли уметь программировать?
Нет. Вы только пишете в чат обычными словами — как в переписке. Код мы не трогаем.
Нужен ли терминал / «чёрное окно с командами»?
Нет. Весь интерфейс — это панель Claude Code внутри VS Code. Терминал для воркшопа не понадобится.
Это платно?
VS Code — бесплатный. Для Claude Code нужна подписка Claude Pro или Max (оформляется на claude.ai) — без неё войти не получится.
Не вижу значок Claude Code после установки.
Перезапустите редактор: CtrlCmd + Shift + P → наберите Reload Window → Enter. И проверьте, что расширение из Шага 2 действительно установлено.
Не пускает войти / просит оплату.
Проверьте, что на аккаунте, которым входите, активна подписка Claude Pro или Max. Если подписка есть, а войти не удаётся — напишите нам заранее.
У меня старый/слабый компьютер — потянет?
VS Code лёгкий и работает почти на любом ноутбуке. Главное — рабочий интернет.
